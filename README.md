# ecommerceBackEnd
##  an application backend using mysql database.  This would be for an e-commerce site and was built using mysql2, express, sequelize, and dotenv.

# installation
## npm i
## mysql -u root -p
## source db/schema.sql
## quit
## npm run seed
## npm start
